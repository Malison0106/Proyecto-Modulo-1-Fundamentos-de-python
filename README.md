# Proyecto-Modulo-1-Fundamentos-de-python

Este programa fue creado de acuerdo a los requisitos del proyecto, tales como:

Un formulario que solicita la información de una o más personas como son: Nombre, edad, estatura y peso. Al capturar estos datos se realiza un cálculo de su índice de masa corporal y arroja el resultado, detallando su estado físico, así como un resumen de dichos datos.

El programa valida las entradas de datos para asegurar que sean correctas y no se introduzcan datos erróneos o incorrectos de acuerdo al dato que se está pidiendo, evitando también que el usuario deje casillas en blanco. Se agregó la verificación de los datos al ser introducidos para evitar que el programa se cierre si se introduce un dato erróneo.

Este bootcamp hasta ahora me ha enseñado poco a poco lo que se puede llegar a hacer en Python, así como poder agregar más opciones, funciones y los usos que se pueden llegar a hacer en Python.
